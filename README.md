# My Python Project

## Setup

1. Clone the repo:
    ```bash
    git clone https://github.com/Nooneyouknowof/FSE_TEAM_3_FINAL_PROJECT.git
    ```

2. Open the project
    ```bash
    cd FSE_TEAM_3_FINAL_PROJECT
    ```
    Or
    ```bash
    code FSE_TEAM_3_FINAL_PROJECT
    ```

3. Install the python dependencies
    ```bash
    pip install -r requirements.txt
    ```
